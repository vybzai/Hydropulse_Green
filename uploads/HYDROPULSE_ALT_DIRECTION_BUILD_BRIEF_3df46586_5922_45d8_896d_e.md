# HydroPulse — Alternate Direction Build Brief

## Objective
Create **one more HydroPulse website direction** with a **clearly different visual approach** from the current prototype.

This version should feel:
- **more colourful**
- **more cheerful / optimistic**
- still **credible and institutionally safe**
- still appropriate for an **EU-funded research project**
- still clearly **premium and intentional**, not generic

This does **not** mean childish, playful, or startup-gimmicky.
The goal is to make the experience feel more alive, inviting, and positive while preserving trust and seriousness.

---

## Key change from the current direction
The current prototype leans more:
- editorial
- restrained
- atmospheric
- muted
- serious

This alternate direction should instead explore:
- a brighter palette
- more visible colour energy
- a more welcoming tone
- a slightly lighter emotional feel
- clearer moments of optimism and accessibility

It should still avoid:
- cheap sustainability clichés
- loud startup gradients
- childish NGO energy
- generic corporate-template design
- anything that feels unserious or visually messy

Think:
> **hopeful, fresh, human, and well-designed**
not
> **corporate grey** or **dark editorial mood**

---

## Structural requirements
The site should still cover the same intended HydroPulse structure:

1. Home
2. About HydroPulse
3. Research Themes / Focus Areas
4. Demonstration Sites / Case Studies
5. Consortium
6. News / Updates
7. Resources / Knowledge Hub
8. Media Kit / Visual Identity
9. Get Involved / Engagement
10. Contact

The agent should reuse the existing synthetic content structure where useful.

---

## Required interaction reference
Please include a **similar design-toggle / direction-switching menu** to the one used in the current prototype.

That means:
- a visible way to switch between design directions or visual presets
- enough flexibility for review and comparison
- a client-friendly preview mechanic
- something that helps present alternative colour / hero / styling choices clearly

It does **not** need to be identical in UI, but it should preserve the same **review utility**.

---

## Logo / brand requirement
Please also account for **logo file changes / logo variant handling**.

The new direction should:
- support different logo applications where appropriate
- work with the provided HydroPulse logo assets
- consider how the logo behaves across different headers / backgrounds / colour treatments
- keep the presentation polished and consistent

The package includes the current brand assets for reference.

---

## Content source
Use the included synthetic content files as the base content model.

In particular:
- `data.js` contains the synthetic HydroPulse project data
- sitemap / structure notes are included separately

---

## Creative guidance
This version should feel like a **genuine alternate candidate**, not just a recolour.

Good directions might include:
- brighter environmental colours used with discipline
- more daylight / open-air / positive landscape energy
- cleaner, friendlier information presentation
- a more optimistic and communicative homepage tone
- stronger use of colour for sectioning and navigation clarity

But it must still feel:
- premium
- trustworthy
- modern
- institutionally credible
- suitable for a 5-year European research project

---

## Deliverable intent
The result should be:
- meaningfully distinct from the current concept
- good enough to compare side-by-side as an alternative direction
- structured enough to reuse the same synthetic content and overall IA
- presentable to a client as a serious design route

---

## Summary
Create a **more colourful, more cheerful, but still credible and premium** alternate HydroPulse website direction using the same content structure, while also including:
- a similar design-toggle review menu
- support for logo variant / logo asset handling
- the same core sitemap and synthetic content model
